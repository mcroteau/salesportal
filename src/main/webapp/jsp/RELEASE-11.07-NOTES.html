*** Randr Inc Web Business Suite - Sales Portal 11.07 (yy.mm.sub) July 2011 ***

There are a lot of changes in this release. If you are upgrading an existing release, don't hesitate to use the Randr Forum and our request page for help. Or we can be reached at 951-369-3427 (US) or john4@randrinc.com

The Salesportal is up on Ubuntu's Launchpad click for the WBS Launchpad page. Select the Salesportal from the list of projects on the lower right of the page.

the Randr Team.

  • BluePrint sp-revenuetracking - Changes to the commission logic to include revenue so you can track proposals and future revenue. Reports added to the Infoportal.
  • BluePrint sp-performancedata - Added revenue and forecast fields into the Prospect table and display screen. You can manually enter revenue. Also added another view with the new fields, Performance Data View to the search prospects results screen. In the future these will be linked to EBS and Order Portal. Includes 2 new graphics. Displaying the performace data is controlled by tags in the appConfig.xml
  • Added fields to prospect table and prospect display for Order Portal and EBS customer numbers to allow for linking between the portals.
  • Search Prospects ordered the result by company name.
  • Installs with $ as a currency code
  • Search prospects screen - added note to use Cltr_click to de-select an option.
  • BluePrint sp-logostandarization - Updated Randr Logo on sign in page and above menu. New graphic - randr-logo-small.jpg
  • BluePrint sp-updateDocfromProspect - We now allow a document description and type to be changed from the prospect screen.
  • BluePrint sp-linktospinsalesactionemail - Added a tag to the appConfig.xml with Salesportal URL for a link in the Sales Action Emails (
  • Added label short name to data base
  • Add user type 3, a user who can search and view prospects only but not download and has no other options
  • Added the ability to control if the heading is Prospect. This allows you to have customers, salesmen, etc as the heading.
  • Added the ability to control the page content of the home page you see when you sign on
