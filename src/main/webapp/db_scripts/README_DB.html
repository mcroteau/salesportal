Sales Portal 10.11

If you are starting from scratch simply deploy the application, and you will be directed automatically to set up your database. You will not need to do anything else. If you are upgrading, please read on...

Randr has included the necessary database correction scripts. They are packaged by version. You can simply upgrade your current database by running the corresponding db_correctionsx.y.z version scripts.

** For example, if you were on version 1.2.5 and wanted to update to version 10.11, you would need to run:

  • db_corrections1.2.6
  • db_corrections1.2.7
  • db_corrections1.2.8
  • db_corrections1.2.9
  • db_corrections1.3.0
  • db_corrections10.05
  • db_corrections10.11

to fully update your database.

** Note : there is a required change in the appConfig.xml for updating your existing installation. Please refer to the UPDATINGAPPLICATION-README.html for more information. **
